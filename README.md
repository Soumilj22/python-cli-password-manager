# CLI Password Manager (Python)

A simple command-line password manager built using Python that securely stores and retrieves account credentials.

## Features
- Add new account credentials
- View stored passwords
- List saved accounts
- Password encryption using Fernet

## Technologies Used
- Python
- Cryptography (Fernet encryption)
- Git

## How to Run

1. Clone the repository

git clone https://github.com/Soumilj22/python-cli-password-manager.git

2. Install required library

pip install cryptography

3. Run the program

python password_manager.py

## Learning Outcomes
This project demonstrates:
- File handling in Python
- Encryption and decryption
- Command Line Interface (CLI) application design
- Basic credential management
